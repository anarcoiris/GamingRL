
# Data Directory

This directory contains game datasets.

## Structure

- `generated/`: Synthetic games generated by bots (JSON format).

## Format

Each JSON file represents a full game:
```json
{
  "game_id": "...",
  "metadata": {"p1": "heuristic", ...},
  "winner": 1,
  "steps": [
    {
       "step_count": 0,
       "current_player": 1,
       "board": [[...]],
       "action": {...}
    }
  ]
}
```
